# Automatic Speech Recognition Bible
Developed by Emmanuel Nwobueze as His undergraduate Project.
built with cmusphinx/sphinx4 library using java programming language.

This project depends on shinx4 speech library to convert speech to text which can then be used to query the database which contains bible passages
